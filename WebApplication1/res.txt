STEP 1: Start SQL Server (Docker)

First, make sure Docker Desktop is running (whale icon 🐳 in the menu bar).

Run SQL Server container

Open Terminal and run:

docker run -e "ACCEPT_EULA=Y" \
  -e "SA_PASSWORD=YourStrong@Password1" \
  -p 1433:1433 \
  --name mssql \
  -d mcr.microsoft.com/mssql/server:2022-latest


If Docker says the container already exists, just start it:

docker start mssql

Verify it’s running
docker ps


You should see mssql in the list.

STEP 2: Connect using Azure Data Studio

Open Azure Data Studio

Click New Connection

Fill in exactly:

Field	Value
Server	localhost
Authentication	SQL Login
User	sa
Password	YourStrong@Password1
Database	(leave blank)
Trust server certificate	✅

Click Connect.

🎉 If it connects, SQL Server is live on your Mac.

STEP 3: Create a Database (Course-style)

In Azure Data Studio → New Query:

CREATE DATABASE AdvDotNetDb;
GO


Refresh → you should see AdvDotNetDb.

STEP 4: Prepare Your MVC Project for EF Core

Go to your MVC project folder:

cd /Applications/dotnet/WebApplication1/WebApplication1

Install EF Core packages
dotnet add package Microsoft.EntityFrameworkCore.SqlServer
dotnet add package Microsoft.EntityFrameworkCore.Tools


Verify:

dotnet list package

STEP 5: Add Connection String

Open appsettings.json:

{
  "ConnectionStrings": {
    "DefaultConnection": "Server=localhost,1433;Database=AdvDotNetDb;User Id=sa;Password=YourStrong@Password1;TrustServerCertificate=True;"
  }
}


Save the file.

STEP 6: Create Your First Model

Create folder:

Models


Add file Student.cs:

using System.ComponentModel.DataAnnotations;

public class Student
{
    public int Id { get; set; }

    [Required]
    public string Name { get; set; }

    public int Age { get; set; }
}

STEP 7: Create DbContext

Create folder:

Data


Add file AppDbContext.cs:

using Microsoft.EntityFrameworkCore;

public class AppDbContext : DbContext
{
    public AppDbContext(DbContextOptions<AppDbContext> options)
        : base(options) { }

    public DbSet<Student> Students { get; set; }
}

STEP 8: Register DbContext

Open Program.cs, add:

builder.Services.AddDbContext<AppDbContext>(options =>
    options.UseSqlServer(builder.Configuration.GetConnectionString("DefaultConnection")));


Also add at the top:

using Microsoft.EntityFrameworkCore;

STEP 9: Run EF Core Migration
dotnet tool install --global dotnet-ef


Then:

dotnet ef migrations add InitialCreate
dotnet ef database update


🎉 Tables created in SQL Server.

Check in Azure Data Studio:

SELECT * FROM Students;

STEP 10: Sanity Check

Run your app:

dotnet run
